# bilibiliReplyTool/哔哩哔哩评论统计分析工具
###version2020815085914###

<h3>使用方法</h3>
<ol>
  <li>下载“bookmarks.html”文件；</li>
  <li>导入“bookmarks.html”到浏览器收藏中；</li>
  <li>用浏览器打开哔哩哔哩视频，点击已收藏的“bilibili评论统计分析工具”标签；</li>
</ol>

<h3>已完成功能</h3>
<ol>
  <li>性别统计；</li>
  <li>评论留言时间段统计；</li>
  <li>年龄统计；</li>
  <li>评论查询；</li>
  <li>用户等级统计；</li>
  <li>牌子用户统计；</li>
</ol>
